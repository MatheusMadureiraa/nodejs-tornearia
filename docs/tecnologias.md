## 🚀 Tecnologias utilizadas

### Backend
- Node.js
- Express
- SQLite3
- Electron (para aplicação desktop)

### Frontend
- HTML
- CSS
- JavaScript

### Banco de dados
- SQLite

### Testes e validação
- Playwright (testes automatizados end-to-end)
- Insomnia (testes e validação de API)

---

## 📁 Estrutura do projeto
- /backend → API e conexão com banco de dados
- /frontend → Interface do sistema
- /backend/database → Configuração e arquivo do SQLite
- /electron → Processo principal do desktop app
